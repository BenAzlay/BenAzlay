## Selected Work

### Amarnai — Open-source AI email triage
Gmail-first AI triage that sorts an inbox into folders defined in plain English, drafts replies for approval, and shows its confidence and reasoning on every routing decision. Built full-stack, solo.
- **Plain-English taxonomy** — describe each folder in a sentence; no rules, no code
- **AI-drafted replies** — generated for review, never sent without approval
- **Confidence + reasoning** — low-confidence threads routed to review instead of guessing
- **Privacy by design** — read-only Gmail OAuth, no email bodies stored
- **Open-source (AGPL-3.0)** — hosted, or self-host with Docker against any model (including local Ollama)
- Next.js, Node/TypeScript, PostgreSQL/Prisma, embedding-based routing, frontier-model integration end to end
→ [amarnai.com](https://amarnai.com) · [github.com/amarnai/amarnai](https://github.com/amarnai/amarnai)
